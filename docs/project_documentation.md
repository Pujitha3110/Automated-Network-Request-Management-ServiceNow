# Automated Network Request Management in ServiceNow

## Introduction

The objective of this project is to automate network service requests using ServiceNow.

Employees can request network-related services through the Service Catalog instead of sending emails or contacting the IT team manually.

---

# Service Catalog

Catalog Name:

**Network Services**

---

# Category

**Network Requests**

---

# Catalog Items

- VPN Access Request
- Wi-Fi Access Request
- Static IP Request
- Firewall Rule Change
- Network Port Activation
- DNS Update Request

---

# Validation

Each catalog item was verified using the **Try It** option in ServiceNow.

All catalog items were successfully created and are accessible from the catalog.
