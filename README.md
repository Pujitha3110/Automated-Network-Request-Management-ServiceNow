# Automated Network Request Management in ServiceNow

## Project Overview

This project automates the lifecycle of network-related service requests using ServiceNow.

Instead of handling requests manually, users can submit requests through the Service Catalog, where approvals, task assignments, notifications, and fulfillment are automated.

## Features

- Service Catalog
- Network Request Forms
- Approval Workflow
- Flow Designer
- Notifications
- Reports and Dashboards

## Technologies Used

- ServiceNow
- Service Catalog
- Flow Designer
- Business Rules
- Notifications
